# rafa.wiki website

A place for me to talk about me and experiment with some new tools.

Made with TypeScript, Next.js, and Tailwind CSS

## Usage

### Install Dependencies
```bash
yarn install
```

### Run Dev Server (http://localhost:3000) 
```bash
yarn run dev
```
